# interactive-graphviz.nvim

Neovim plugin scaffold for a live Graphviz/DOT browser preview.

v1 scope is intentionally small:

- command-started preview lifecycle
- live browser rendering through bundled Graphviz-WASM
- `dot` and `neato` layout selection
- zero-prerequisite install on supported platforms

Deferred: auto-open, export, additional engines, and richer browser interactivity.
